
# TitleIQ Agent Streamlit App

This app provides a Streamlit interface for the existing four-phase TitleIQ Agent workflow.

## Files
- `streamlit_app.py` — Streamlit interface
- `titleiq_master_orchestrator.py` — existing integrated analysis logic
- `requirements.txt` — packages for local run / Streamlit Cloud

## Local run
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
